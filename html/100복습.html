단축키

단축키 p, h1 후 탭 하면 태그 자동 생성

h$*0 = 0있는곳에 숫자를 넣으면 h1부터 차례대로 h1,h2,h3 ... 추가됨.
h$*n{설명}괄호 안에 문자를 넣으면 n개 만큼{}안에 설명 생성됨

p*n p단락이 n개 만큼 생성됨.

태그창에 F2번을 누르면 처음과 끝 태그를 수정 할 수 있음.

주석 단축키는 Ctrl+/

도구창 단축키는 Ctrl+space bar

>표시는 큰거 안에 작은거 삽입 할 수 있음. ex) html>head+body를 치면 html태그 안에 head태그 + body태그 생성

html:5 입력 후 Tab키 누르면 HTML자동 셋팅

h1 = 제목 h2 = 다음 중요한 내용 ... h6까지 있음 1~6까지 숫자가 커질수록 제목이 작아짐.
title은 탭에 나타날 제목
br = 줄 띄어쓰기
hr = 페이지에 가로줄이 생김 (페이지안에서 분리)
