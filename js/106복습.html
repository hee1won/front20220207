querySelector

우선 javascript는 dom (document object model) dom은 우리가 작성한 html, css 문서를 객체로 만들어서 올린다. 그리고 다음 배운 내용은 quarySelector이다. quarySelector는 선택자이고 quarySelector를 이용해 html이나 css 파일등을 고치거나 추가 할 수 있다. querySelector는 주로 태그를 선택자로 한다. (필요에 따라 id 또는 class를 사용 할 수 있음.) ex > quarySelector를 이용해서 버튼을 클릭하면 텍스트가 나오게 하는 법.
