38.boolean 참 거짓

boolean은 참 거짓을 판별한다.
예를들어 console.log(true)는 참, console.log(false)는 거짓.
let a = true;
let b = false
console.log(a)는 결과값이 나오지만 console.log(b)는 거짓이라 결과가 나오지 않는다.

48.논리 연산자

논리 연산자는 && (AND), ||(OR), !(NOT)연산자 등이 있다.

&& AND연산자.

피연산자가 두개,
피연산자의 type : boolean
연산결과 : boolean
결과 : 두피연산자가 모두 true일 때만 true 나머지는 모두 false

|| OR 연산자.

피연산자가 두개
피연산자의 type : boolean
연산결과 : boolean
결과값 : 두피연산자가 모두 false일 때만 false, 나머지는 모두 true

! NOT연산자.

피연산자가 하나
피연산자의 type : boolean
연산결과 : boolean
결과값 : 피연산자가 true이면 false, 피연산자가 false 이면 true

52.삼항 연산자



삼항 연산자 a ? b : c 결과값 : a가 true면 b, a가 false면 c

ex)
console.log(true ? 1 : 2); 값은 1

console창에 값이 true면 1 false면 2를 출력하라. 값 : true


console.log(false ? 1 : 2); 값은 2

console창에 값이 true면 1 false면 2를 출력하라. 값 : false
